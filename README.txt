To use this program run: 
```
java Roulette
 
 ```

 The program will display the first name of the player from the players.txt. Then enter a bet (1-36 or EVEN or ODD), space and finally enter the amount of 
 money you are betting with. 
 After hitting ENTER, the program will ask if you want to make another bet, if yes enter 'yes' and repeat the steps above.

 If you want to stop playing enter 'no'.

 The program will the display the winnings of the first player in the players.txt and then the second player will have a go.
